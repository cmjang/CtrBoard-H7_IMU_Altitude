Capabilities

This driver supports USB High, Full and Low speeds.

Instances

Hardware resource relating to driver instance is shown in the table below:

Driver Instance Hardware Resource
Driver_USBH0 USB_OTG_HS

Configuration

Compile-time

Definitions used for compile-time configuration of this driver are shown in the table below:

Definition Default value Max value Description
USBH0_MAX_PIPE_NUM 16 16 Defines maximum number of Pipes that the driver will support. This value impacts driver memory requirements.
USBH0_BULK_IN_NAK_DELAY 0 65535 Delay between NAKs on the Bulk IN pipe (in SOFs), value 0 = not used (no delay between NAKs)

STM32CubeMX

The USB Host driver requires:

  • USB peripheral clock at 48 MHz
  • USB_OTG_HS peripheral configured as Host_Only
